# EMI Final Project Idea

## 项目名称

**艺术家风格检索与参考板工具**  
Artist Style Retrieval and Reference Board Tool

## 这个项目是干什么的

这个项目要做的是一个**帮助创作者找参考图和整理 moodboard 的工具**。

用户可以上传一张图片，或者输入一个简短的创作方向描述，例如：

- 冷静、克制、建筑感强
- 戏剧化、历史感、暗色调
- 柔和、有机、植物感

系统会从一个参考图库里找出风格更接近的图像，帮助用户继续寻找视觉方向。

当前初版会以几位公开艺术家作品作为参考库来实现，例如：

- Monet
- Van Gogh
- Hokusai
- Klimt

所以它更像一个“艺术家参考检索工具”，而不是风格分类器。

## 它能做什么

这个项目完成后，主要会有这些功能：

1. 用户上传图片作为查询
2. 或者输入一个文字 brief 作为查询
3. 系统返回一组视觉上更接近的参考图
4. 用户可以根据这些结果继续整理自己的参考板和灵感库

## 它和普通图片搜索有什么不一样

它不是普通的关键词搜索，而是一个带有机器学习能力的**视觉相似性检索工具**。

也就是说，它不仅看文件名或标签，还会根据图像的整体视觉特征来寻找更接近的参考图。

## 为什么这个题目适合作业

这个题目适合作为 EMI 的最终大作业，因为它：

1. 有明确的机器学习内容
2. 和创作实践直接相关
3. 展示形式直观，容易录视频
4. 可以讨论机器推荐如何影响创作者寻找灵感的方式

## 项目的核心想法

这个项目想探索的不是“机器能不能定义艺术”，而是：

**机器学习能不能帮助创作者更快地找到视觉参考，同时也暴露出算法推荐本身的偏向和局限。**

## 最后会交付什么

最后会形成一个完整的课程项目包，包括：

1. 公开 GitHub 仓库
2. 可运行代码
3. 参考图像数据集和来源说明
4. 检索模型与索引构建代码
5. 一个可演示的网页小工具
6. README
7. weblog
8. 3 到 5 分钟的视频演示

## 一句话总结

这是一个把**图像检索、创作参考整理、机器学习和批判性思考**结合起来的课程项目。  
它更像一个灵感搜索工具，而不是一个风格分类器。
