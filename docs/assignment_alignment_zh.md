# 项目与作业要求对齐说明

这个项目与 EMI 最终作业要求的对应关系如下：

## 1. 机器学习实现

项目不是只调用现成网页工具，而是包含完整代码流程：

1. 参考图库整理
2. metadata catalog 构建
3. CLIP embedding 提取
4. 检索索引构建
5. 检索评估
6. Gradio demo

## 2. creative practice 相关性

项目直接服务于创作者的参考图搜索和 moodboard 整理：

1. 帮助用户上传图片找相近艺术参考
2. 支持用文本 brief 搜索视觉方向
3. 讨论推荐系统如何影响创意研究过程

## 3. 课程中的机器学习主题

项目能体现：

1. 数据准备和来源记录
2. 预训练模型选择
3. embedding-based retrieval 方法
4. 基础评估与 case study
5. 对算法相似性的批判性反思

## 4. 过程记录要求

仓库中已经包含：

1. `weblog.md`
2. 数据和 manifest
3. 索引构建脚本
4. 评估报告
5. demo 界面

## 5. 视频展示要求

这个项目适合录制 3-5 分钟视频，因为可以直接展示：

1. 数据库结构
2. 图像查询结果
3. 文本 brief 查询结果
4. 检索排行和相似度

## 6. 批判性思考空间

这个项目适合讨论：

1. 算法如何定义“相似”
2. 推荐系统是否会收窄创作视野
3. 数据集选择如何影响结果
4. 机器学习如何塑造灵感搜索路径
