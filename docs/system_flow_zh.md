# 整体流程逻辑

## 1. 数据准备

项目先整理一个独立艺术家参考库：

1. Monet
2. Van Gogh
3. Hokusai
4. Klimt

## 2. catalog 构建

系统扫描图片目录，为每张图记录：

- 路径
- 标题
- 作者
- 来源
- brief 提示词

## 3. 特征提取

使用 CLIP 模型把：

- 图像
- 文本 brief

都映射到同一个向量空间。

## 4. 检索排序

当用户上传图片或输入文本时：

1. 系统计算查询向量
2. 和参考图库向量做相似度比较
3. 返回 top-k 结果

## 5. 展示结果

demo 会输出：

1. 查询说明
2. 相似参考图 gallery
3. 排序表格
4. brief hint 和 creator 信息
