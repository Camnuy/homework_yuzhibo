# 项目出版版简介

## 项目名称

艺术家风格检索与参考板工具  
Artist Style Retrieval and Reference Board Tool

## 这是一个什么项目

这是一个面向创作者的视觉参考检索工具。

用户可以：

1. 上传一张图片作为查询
2. 输入一段简短的创作 brief 作为查询
3. 让系统返回一组视觉上更接近的艺术参考图

它更像一个 **moodboard / reference board 辅助工具**，而不是一个风格分类器。

## 当前版本的核心思路

这个项目的重点是：

- 用机器学习帮助创作者更快找到视觉参考
- 通过图像 embedding 做相似图检索
- 让“找灵感”这件事更有方向感

## 当前独立数据集

这一版使用的是一个**独立艺术家参考库**，包含四组公开艺术家作品：

1. `Monet`
2. `Van Gogh`
3. `Hokusai`
4. `Klimt`

这套数据与另一个分类项目完全分开，不共用图片。

## 当前能展示什么

当前初版已经可以展示：

1. 图像查询
2. 文本 brief 查询
3. 排序后的参考图结果
4. 检索相似度信息

## 当前版本结果

当前 starter 版本使用 32 张独立参考图，已有一轮初步检索评估结果：

- Top-1 label agreement: `0.8125`
- Top-3 label agreement: `0.9375`

这说明它已经具备一个可演示的初版形态。

## 这个项目适合作业的原因

它符合 EMI 作业要求，因为它同时具备：

1. 机器学习实现
2. creative practice 场景
3. 可运行 demo
4. 可记录的研究与开发过程
5. 可讨论的批判性反思

## 一句话总结

这是一个用机器学习帮助创作者检索艺术参考图、构建 moodboard 的课程项目初版。
